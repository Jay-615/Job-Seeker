# Job-Seeker

A personal job-search workflow tool built as a Claude Code workspace. Clone the
repo, run `claude` in the folder, and use a small set of slash commands to find
new postings across LinkedIn, Indeed, Dice, and selected company career pages;
score them on experience match, network proximity, and target-company alignment;
generate tailored resumes and cover letters for the best matches; and track
outcomes so the same jobs don't resurface. Designed to keep the human in the
loop — the tool stops at "tailored draft ready" and you submit applications
yourself.
